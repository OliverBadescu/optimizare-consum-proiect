Tema Proiect:
  Optimizarea consumului de combustibil al unui vehicul autonom
  
Descriere: Se cere dezvoltarea unui algoritm care optimizează traiectoria unui vehicul autonom astfel încât să reducă consumul de combustibil.

Detalii implementare:
  • Modelarea traseului vehiculului pe baza unor puncte de interes.
  • Utilizarea algoritmilor A* sau Dijkstra pentru optimizarea rutei.
  • Simulare MATLAB și comparație cu rute neoptimizate.

Task 1: Modelarea matematică a consumului de combustibil și identificarea factorilor de influență.

Task 2: Analiza complexității algoritmilor de optimizare a traseului.

Task 3: Structuri de date utilizate pentru reprezentarea traseului vehiculului.

Task 4: Algoritmi exacti și metaeuristici pentru optimizarea traseului.

Task 5: Implementarea și testarea în MATLAB.

Task 6: Compararea performanțelor algoritmilor.

Task 7: Aplicații reale în transport și vehicule autonome.

Rezultate așteptate:
  • Reducerea consumului de combustibil prin trasee optimizate.
  • Analiză a timpului de execuție și a distanței parcurse.
